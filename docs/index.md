# ZenithAFK Documentation

The ultimate AFK zone plugin for modern Paper servers.

---

## Why ZenithAFK?

ZenithAFK is a fully-featured, high-performance **AFK zone plugin** built for Paper 1.20+ and Folia. Create configurable AFK zones where players earn **periodic rewards** just by standing inside, with a powerful **Hard-AFK idle detection** system, an in-game **GUI editor**, per-zone customization, **playtime tracking**, and full PlaceholderAPI support.

No dependencies required beyond PlaceholderAPI (optional). Vault economy integration is **auto-detected** — no config needed.

---

## Key Features

- :material-selection: **Unlimited AFK Zones** — WorldEdit-like selection wand, per-zone config
- :material-gift: **Weighted Rewards** — probability-based reward tables with commands, items, or Vault money
- :material-fire: **Streak & Combo** — escalating rewards the longer players stay
- :material-shield-lock: **Anti-Abuse Suite** — anti-macro, anti-redirection, daily limits, auto-kick
- :material-clock-outline: **Time-Based Zones** — schedule active days/hours, enter conditions
- :material-earth: **Multi-Server Sync** — MySQL cross-server player data with periodic refresh
- :material-eye: **Hard-AFK Detection** — idle detection, forced teleport, full protection suite
- :material-chart-bar: **Playtime Tracking** — built-in leaderboards, per-zone time, PlaceholderAPI
- :material-monitor-screenshot: **Dynamic Displays** — bossbar mm:ss countdown, titles, action bar
- :material-wrench: **In-Game GUI Editor** — edit everything without touching YAML
- :material-cash: **Vault Economy** — auto-detected, deposit money rewards seamlessly
- :material-update: **Auto-Update Checker** — BuiltByBit integration, notifies admins on join
